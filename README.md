## Music Track Searcher FrontEnd App

Autor: Rafael Rodríguez Juárez

A frontend application developed with React used to filter data (Artists, Albums and Tracks) from Spotify API.
React Version 17.0.2
React Router 5.2.0
Bootstrap 5.0.0 

Installation:

## `yarn install`  

To Start Server:

## `yarn start`  

To Visit App:

## `localhost:3000/`
